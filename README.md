## English

Non-official Chrome/Edge MV3 extension that improves browsing tender lists on
ezamowienia.gov.pl by adding real links, filters, and productivity tools.

## Features

- Direct "Open" links on tender lists and BZP notice links
- Optional row click, middle-click, and background-tab opening
- Quick filters (status, city, organization) plus starred offers
- Compare selected offers side-by-side
- Keyboard shortcuts, quick jump search, and next-unread navigation
- Sticky toolbar, continue-reading panel, and page progress indicator
- Copy ID/link actions, inline details, status badges, and notes
- Polish/English UI with auto-detect

## Install (developer mode)

1. Open `chrome://extensions` or `edge://extensions`
2. Enable Developer mode
3. Click "Load unpacked" and select this folder

## Options

Use the extension options page to configure behavior, UI, filters, and language.

## Permissions

- `storage`: save settings and filters
- `tabs`: open offers in new/background tabs
- `https://ezamowienia.gov.pl/mp-client/*`: modify list pages

## Polski

Nieoficjalne rozszerzenie Chrome/Edge (MV3), które usprawnia przeglądanie list
postępowań na ezamowienia.gov.pl poprzez dodanie prawdziwych linków, filtrów
i narzędzi produktywności.

## Funkcje

- Bezpośrednie linki "Open" w tabelach oraz linki do ogłoszeń BZP
- Opcjonalne otwieranie po kliknięciu w wiersz, środkowym przyciskiem i w tle
- Szybkie filtry (status, miasto, organizacja) oraz oferty oznaczone gwiazdką
- Porównywanie wybranych ofert obok siebie
- Skróty klawiaturowe, szybkie wyszukiwanie i nawigacja po nieprzeczytanych
- Pływający pasek narzędzi, panel kontynuacji i wskaźnik stron
- Kopiowanie ID/linku, szczegóły w wierszu, znaczniki statusu i notatki
- Interfejs PL/EN z automatycznym wykrywaniem

## Instalacja (tryb deweloperski)

1. Otwórz `chrome://extensions` lub `edge://extensions`
2. Włącz tryb deweloperski
3. Kliknij "Load unpacked" i wskaż ten folder

## Opcje

Strona opcji pozwala skonfigurować zachowanie, UI, filtry i język.

## Uprawnienia

- `storage`: zapis ustawień i filtrów
- `tabs`: otwieranie ofert w nowych/tylnych kartach
- `https://ezamowienia.gov.pl/mp-client/*`: modyfikacja stron list
