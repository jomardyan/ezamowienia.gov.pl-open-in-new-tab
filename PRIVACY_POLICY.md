# Privacy Policy

Last updated: 2025-12-29

This is a non-official browser extension for ezamowienia.gov.pl. It does not
collect, sell, or transmit personal data.

## Data the extension handles

- Page content: read from ezamowienia.gov.pl list pages to add links and UI.
- Settings: stored in `chrome.storage.sync`.
- Filter presets: stored in `chrome.storage.local`.
- Session state (opened IDs, scroll position): stored in `sessionStorage`.
- Clipboard: used only when you click a copy button.

## Data sharing

No data is sent to third parties or external servers.

## Permissions used

- `storage`: save settings and filters.
- `tabs`: open offers in new/background tabs.
- `https://ezamowienia.gov.pl/*`: run the content script on the site.

## Your control

You can clear extension data from your browser settings or uninstall the
extension at any time.
